# NBP-currencies

Simple web user interface which connects with NBP web API (​http://api.nbp.pl/)​

## Requirements
```js
Node >= 10.13.0 
````

## Installing
```js
npm install
```

## Running
```js
npm start
```

## Building for production
```js
npm run build
```

## Testing
```js
npm run test
```

## Update snapshot tests
```js
npm run updateSnapshot
```
